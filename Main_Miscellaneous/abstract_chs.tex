\begin{chineseabstract}
% \footnotetext{*本研究得到某某基金（编号：）的资助}

基于 Wi-Fi 信道状态信息（CSI）的人类活动识别技术凭借非接触、低成本与隐私保护等优势，已成为智能感知领域的重要研究方向。CSI 能够反映人体运动引起的无线信道多径变化，为室内动作识别提供有效依据。然而，现有方法通常假定训练阶段可获得全部类别数据，难以适应实际应用中类别持续扩展、旧类数据难以重复获取以及新增类样本有限等问题。围绕上述挑战，本文面向 Wi-Fi CSI 室内动作识别中的持续学习需求，分别针对新增类样本相对充足和新增类样本极少两类典型场景开展研究。

针对新增类样本相对充足的类增量学习场景，本文提出一种回放—扩展—巩固框架。该方法以记忆库重放为基础，通过选取旧类代表样本缓解增量阶段的新旧类分布失衡；在此基础上，采用冻结骨干与分类器扩展相结合的方式实现新类别接入，在尽量保持旧类表征稳定的同时增强模型对新类的适应能力；进一步结合教师—学生结构与双边知识蒸馏，对扩展阶段形成的新旧类知识进行统一巩固。针对新增类样本极少的少样本类增量学习场景，本文提出一种基于时序增强与图注意力的学习方案。该方法围绕 CSI 时序信号建模，设计多尺度时序编码器提取不同时间尺度下的动作模式，引入时序注意力融合模块聚合关键信息，并在增量阶段利用增强图注意力网络刻画类别原型之间的关系；同时，通过双分类器架构对不同来源的判别信息进行加权融合，以缓解少样本条件下的新旧类偏置与类别级过拟合问题。在 XRF 数据集上的实验结果表明，所提类增量学习方法在多阶段类别扩展过程中能够保持较好的整体识别性能，末次增量阶段准确率达到 73.57\%；所提少样本类增量学习方法在 5-way 1-shot 设定下同样表现出较好的稳定性，末次增量会话准确率达到 74.24\%，平均准确率达到 78.48\%，整体优于对比方法。

综上，本文围绕 Wi-Fi CSI 室内动作识别在类别持续扩展条件下面临的灾难性遗忘与样本稀缺问题，分别从类增量学习和少样本类增量学习两个层面提出了相应方法，并通过实验验证了所提方法的有效性。相关研究为面向动态场景的 Wi-Fi 感知系统持续更新提供了可行思路，也为后续开展更高鲁棒性和更强泛化能力的无线感知研究奠定了基础。
